# AviUtl2-MyAssetManager

AviUtl2用 アセット（素材）管理プラグイン v1.0
作成者: [なたでここ/Natadecoco_2539]

■ 概要
AviUtl2で画像や動画素材、オブジェクトの構成を便利に管理・再利用するためのプラグインです。
本プラグインのアセット保存・管理には、AviUtl2のエイリアス機能を利用しています。

1. 直感的なプレビュー: 一覧の素材にマウスカーソルを合わせるだけで、GIFや画像を即座にプレビュー再生します。
2. ドラッグ＆ドロップ対応: 管理画面からタイムラインへ直接素材を放り込むことができます。
3. キャプチャ登録: 画面上の任意の範囲をキャプチャし、そのままアセットのプレビュー画像として登録可能です。
4. 柔軟な整理: カテゴリ分けや検索機能、★お気に入り登録（優先表示）により、膨大な素材をスマートに管理できます。

■ インストール方法
このフォルダに入っている `MyAssetManager.aux2` を AviUtl2の「Plugin」フォルダに入れてください。

■ 使い方
1. **メイン画面の表示**: メニューの「表示」→「My Asset Manager」を選択して管理ウィンドウを表示します。
2. **素材の追加**: タイムライン上でオブジェクトを選択し、右クリックメニューの「プラグイン」→「MyAsset: 追加」を選択します。
3. **単独エディタの表示**: タイムライン上のオブジェクトを右クリックして「MyAsset: 編集」を選択すると、その素材専用の編集ウィンドウを個別に開くことができます。
4. **プレビュー設定**: 管理画面の右クリックメニュー「プレビュー設定...」から、プレビュー再生速度の倍率（50%〜1000%）を調整し、PCへの負荷を最適化できます。

■ 開発者向け情報
本プラグインは Visual Studio 2022 (C++17) 環境でビルドされています。
`build_msvc.bat` を実行することで、ソースコードから `.aux2` ファイルを生成可能です。
